# more0ai/common (PHP)

Shared common layer for more0ai: envelope types, pipeline, middleware, policy, wire protocol, bootstrap, worker assignments.

## Install

```bash
composer require more0ai/common
```

## Status

Scaffold. Implementation to follow (mirror Node SDK). Client and worker PHP SDKs depend on more0ai/common.
